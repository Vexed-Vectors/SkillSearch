# Python MCP Server Skill

Guides GitHub Copilot in building Model Context Protocol (MCP) servers in Python — proper tool definitions, resource handlers, and server lifecycle management.

## What This Skill Does

When active, Copilot will understand MCP server architecture and generate code following the official Python MCP SDK conventions — including tool definitions with proper docstrings, resource handlers, structured error handling, and project layout.

## Key Areas Covered

- MCP server setup with the official Python SDK
- Tool definitions with proper typing and docstrings
- Resource and resource template handlers
- Error handling patterns for MCP servers
- Recommended project structure
- Testing strategies for MCP tools

## Usage

Place `instruction.md` at `.github/copilot-instructions.md` in your MCP server project workspace.
