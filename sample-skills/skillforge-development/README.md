# SkillForge Development Skill

Copilot instructions for developing the SkillForge platform — an internal tool for managing and distributing AI copilot skill files across engineering teams.

## What This Skill Does

When active, GitHub Copilot will understand the SkillForge project architecture, coding conventions, data model, and API design to provide contextually relevant code suggestions and assistance.

## Key Areas Covered

- **Monorepo structure**: Registry API, VS Code Extension, Web Portal, CLI
- **Tech stack**: Node.js, Express, SQLite, TypeScript, Next.js, Commander.js
- **Data model**: Skills, Packages, Personas, and their relationships
- **API design**: RESTful endpoints for skill/package CRUD and search
- **Coding guidelines**: File conventions, error handling, module patterns

## Usage

Place `instruction.md` at `.github/copilot-instructions.md` in the SkillForge workspace.
