# React TypeScript Components Skill

Instructs GitHub Copilot to generate React components using TypeScript best practices — proper typing, hooks patterns, component composition, and accessible UI.

## What This Skill Does

When active, Copilot will follow modern React + TypeScript patterns including proper prop typing, custom hook conventions, CSS Modules for styling, accessibility best practices, and state management guidelines.

## Key Areas Covered

- Functional component patterns with TypeScript
- Custom hooks conventions
- State management (Context + useReducer vs. local state)
- CSS Modules for styling
- WCAG accessibility requirements
- Error boundary patterns

## Usage

Place `instruction.md` at `.github/copilot-instructions.md` in your React workspace.
