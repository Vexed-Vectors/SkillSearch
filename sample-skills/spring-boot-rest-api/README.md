# Spring Boot REST API Skill

Guides GitHub Copilot to generate idiomatic Spring Boot REST APIs with proper exception handling, validation annotations, and layered architecture patterns.

## What This Skill Does

When active, Copilot will follow established Spring Boot conventions including layered architecture (Controller → Service → Repository), proper DTO usage, Jakarta Validation, global exception handling, and RESTful API design patterns.

## Key Areas Covered

- Layered architecture enforcement
- REST endpoint conventions and HTTP status codes
- Jakarta Validation on DTOs
- Global exception handling with `@RestControllerAdvice`
- Java 17+ features (records, sealed classes)
- Naming conventions for all layers

## Usage

Place `instruction.md` at `.github/copilot-instructions.md` in your Spring Boot workspace.
